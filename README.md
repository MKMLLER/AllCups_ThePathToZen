# AllCups. Соревнование с заключительного этапа всероссийской олимпиады по ИИ 2024 года. Путь к дзену.
# Задача в тренажере: https://cups.online/ru/training/8/tasks/2461

Суть решения:
Решение задачи осуществлено на нескольких моделях, для генерации эмбеддингов используется модель rubert-tiny2:
		- Дерево решений, обученное на косинусном сходстве эмбеддингов предложений
		- CatBoost, обученный на косинусном сходстве
		- CatBoost, обученный на косинусном сходстве, евклидовом расстоянии, произведении Адамара
		- С помощью графа, вершины которого - все Id текстов, были получены фичи: степень узла, PageRank,
		  Мера Адамик-Адара, мера Жаккара. На них обучился Catboost, который выдал лучшую метрику из всех решений:
		  0.4886.
		- fine-tuning на двух эпохах предобученого трансформера ruBert. Но он выдал самый худший результат. Такое могло произойти 		  из-за маленького различия между соседними классами таргета. Например, среднее косинусное сходство двух текстов класса
		  Relevant - 0.815, а relevant_minus - 0.795. Так же распределение четырех классов в train не одинаковое, при обучении в
		  CrossEntropyLoss использовался параметр weight.
